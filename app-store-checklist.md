# 📋 App Store 发布检查清单

## ✅ 准备阶段

### 账号和工具
- [ ] Apple Developer 账号已注册（$99/年）
- [ ] Mac 电脑可用
- [ ] Xcode 已安装（最新版本）
- [ ] 已阅读 Apple 审核指南

### 开发准备
- [ ] 应用已用 Capacitor 转换为 iOS 应用
- [ ] 代码已测试，无严重 Bug
- [ ] 性能已优化（启动时间 < 3秒）
- [ ] 支持多种 iPhone 屏幕尺寸

### 基本信息
- [ ] Bundle Identifier 已设置（例如：com.shanghaitour.guide）

### 权限配置

## 📄 必需内容

### App Store Connect 信息
- [ ] 应用名称（最多 30 个字符）
- [ ] 副标题（可选，最多 30 个字符）
- [ ] 应用描述（详细说明功能）
- [ ] 关键词（最多 100 个字符）
- [ ] 分类已选择
- [ ] 支持 URL（你的网站或帮助页面）
- [ ] 营销 URL（可选）
- [ ] **隐私政策 URL**（必需！）

### 截图
- [ ] 6.7" iPhone 截图（1290 x 2796）
- [ ] 6.5" iPhone 截图（1242 x 2688）
- [ ] 5.5" iPhone 截图（1242 x 2208）
- [ ] 至少准备 3 张截图（最多 10 张）
- [ ] 截图展示核心功能

### 隐私政策
- [ ] 隐私政策页面已创建
- [ ] 隐私政策 URL 可访问
- [ ] 说明收集哪些数据
- [ ] 说明如何使用数据
- [ ] 说明数据存储位置
